# ✋ 手指可動域（ROM）自動解析・管理システム

理学療法における「手指の関節可動域（ROM）測定」をデジタル化（DX）し、リハビリの経過を定量的に可視化するためのWindows専用デスクトップアプリケーションです。

Python環境や面倒な設定は一切不要です。配布している **`.exe` ファイルをダブルクリックするだけ**で、AI（MediaPipe）による手指骨格検知システムをすぐにお試しいただけます。

---

## 🖥️ 操作画面イメージ（対話型インターフェース）

<img width="1722" height="1030" alt="image" src="https://github.com/user-attachments/assets/8315f320-42be-4319-bccb-703bb03c2b9b" />

プログラミングやITの知識がない現場の医療スタッフ様でも迷わず使えるよう、「画面の指示に従って数字（1または2）を入力するだけ」で全ての解析が完了する、シンプルで親切な設計（対話型CUI）を採用しています。

- **患者ID・氏名管理機能**: 測定データを個人ごとに識別して蓄積・保存可能。
- **2つの計測モード**: PCのWebカメラを使った「リアルタイム計測」と、スマホ等で「過去に撮影した動画ファイルからの計測」の両方に対応。

動画の解析後は、システムが自動で最大屈曲・最大伸展角度を抽出し、リハビリの回復経過を折れ線グラフとして自動作成・出力します。

## 🚀 主な機能

- **超簡単・即時実行（.exe化）**: 面倒なPython環境の構築（環境構築エラー等）に悩まされることなく、どなたでも今すぐ起動できます。
- **リアルタイム骨格検知**: Webカメラや撮影動画から、手指の関節位置（21拠点）をAIが自動検出。
- **角度自動計算**: 選択した関節（PIP関節など）の屈曲・伸展角度をフレーム単位でリアルタイムに数理計測。
- **経過推移の自動グラフ化**: 過去の測定データと今回の結果を組み合わせ、自動でグラフ画像を出力。

## 📦 起動・実行方法（Windows専用）

1. 本リポジトリから `手指ROM測定システム.exe`（または圧縮されたZIPファイル）をダウンロードします。
2. ダウンロードした `.exe` ファイルをダブルクリックして起動します。
3. 画面に表示される案内に沿って、患者IDを入力し、モード（1または2）をキーボードで入力するだけで解析がスタートします。
